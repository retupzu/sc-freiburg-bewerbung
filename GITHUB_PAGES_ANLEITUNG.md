# Website kostenlos öffentlich machen

## Empfohlene Lösung: GitHub Pages

Für deine Website ist GitHub Pages die einfachste kostenlose Lösung, weil deine Seite nur aus `HTML`, `CSS`, `JS` und Bildern besteht.

## So gehst du vor

1. Erstelle ein kostenloses Konto auf GitHub.
2. Lege ein neues Repository an, zum Beispiel `sc-freiburg-bewerbung`.
3. Lade alle Dateien aus diesem Ordner hoch:
   - `index.html`
   - `styles.css`
   - `script.js`
   - den kompletten Ordner `assets`
   - `.nojekyll`
4. Öffne auf GitHub dein Repository.
5. Gehe zu `Settings` -> `Pages`.
6. Wähle unter `Build and deployment`:
   - `Source: Deploy from a branch`
   - `Branch: main`
   - `Folder: / (root)`
7. Speichere die Einstellung.
8. Nach kurzer Zeit ist deine Seite online. Laut GitHub kann die Veröffentlichung nach dem Push bis zu 10 Minuten dauern.

## Dein Link

Wenn dein GitHub-Name zum Beispiel `kassemjaffal` ist und dein Repository `sc-freiburg-bewerbung` heißt, wird der Link meistens so aussehen:

`https://kassemjaffal.github.io/sc-freiburg-bewerbung/`

Genau diesen Link kannst du dann in deinen Lebenslauf schreiben.

## Beste Stelle im Lebenslauf

Am saubersten ist diese Zeile direkt im Kontaktblock:

`Bewerbungswebsite / Portfolio: https://kassemjaffal.github.io/sc-freiburg-bewerbung/`

## Wichtiger Hinweis

Wenn du einen kürzeren Link willst, musst du das Repository genau so nennen wie deinen GitHub-Namen plus `.github.io`, also zum Beispiel:

`kassemjaffal.github.io`

Dann ist dein Link später noch kürzer:

`https://kassemjaffal.github.io/`
